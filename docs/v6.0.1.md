# v6.0.1 — "RefineMatters"

A refinement release: a broad polish, hardening and correctness pass across the
whole product, a batch of UI fixes, and two new alerts. No breaking changes —
upgrading is a straight swap of the server code and (optionally) the agent.

## Fleet & interface

- **Sidebar reorganised.** Virtualization and Containers moved under **Fleet**,
  Integrations under **Monitoring**, and the App catalog under **Automation**.
  Every sidebar sub-menu is now sorted alphabetically so pages are easier to find.
- **Real world map** on the Sites page — actual coastline geography instead of the
  previous stylised outline. Sites with coordinates still plot by rolled-up health.
- **Single-device auto-patch.** A New auto-patch policy can now target one device
  directly, in addition to a group, tag, site or the whole fleet.
- **App catalog** now renders as a full, sortable table; **Services** gains a
  dedicated **Service baselines** editor card (previously only reachable from a
  device drawer); the **Patch report** export adds **PDF** next to CSV and XML.
- Consistent button sizing (no more buttons spanning a whole card), tidier and
  aligned table action buttons, and Server-status cards that line their values up
  in a single column.

## Alerts & monitoring

- **Certificate expiry raises an alert** by default now (coalesced to one alert per
  host), not just a needs-attention item — with an explicit opt-out in Settings.
- **New alert — read-only remount.** A local filesystem the kernel flips read-only
  (a silent data-loss outage) now reaches the Alerts inbox and your webhooks.
- **New alert — mail-queue backlog.** A host whose mail queue crosses its threshold
  now raises an alert.
- Two alert-resolution fixes: recovering one watched process or one storage pool no
  longer clears the still-open alerts for the others on the same host.

## Under the hood

- Extra defense-in-depth on outbound vulnerability-database and image-registry
  lookups, and a read-only role can no longer clear a ticket's unread badge.
- Faster heartbeats — several per-request maintenance checks and report builders no
  longer copy whole data stores on the common path.
- Documentation refreshed to match the shipping product; typography and spacing
  standardised across pages. See the published [security review](security-review-6.0.1.md).
