# RemotePower v3.1.0 — Release Notes

**Released:** 2026-05-26

## Overview

v3.1.0 is a major UX release focusing on sidebar navigation clarity and
strict Content-Security-Policy compliance. No schema changes. No migrations
required.

## Upgrade

```
make dist          # builds dist/remotepower-3.1.0.tar.gz
```

Deploy the tarball, reload nginx, then hard-reload the browser once to
evict the old service-worker cache (`remotepower-shell-v3.0.6` →
`remotepower-shell-v3.1.0`).

## What's New

### Focused subcategory views

Each sidebar sub-item now shows **only its own content** when clicked.
Previously, clicking e.g. "Device Metrics" in the Monitoring group still
rendered all four monitoring panels (Targets, Metrics, Ports, Scripts).

Affected groups and their sub-items:

| Group | Sub-items |
|-------|-----------|
| Monitoring | Targets · Device Metrics · Listening Ports · Custom Scripts |
| Containers | Docker/Podman/K8s · Proxmox LXC |
| Security | TLS/DNS Expiry · ACME Certificates |

Navigating via keyboard shortcut (`g` → key) or the command palette still
shows all panels together for a full overview.

When navigating directly to **Proxmox LXC** on a node that isn't configured
yet, a "not configured" hint is shown instead of a blank screen.

### TLS/DNS Expiry and ACME Certificates split

The single "TLS / DNS" sidebar button is now two separate sub-items under
Security:

- **TLS / DNS Expiry** — server-side certificate and DNS watchlist
- **ACME Certificates** — Let's Encrypt / ACME managed certificates

### Collapsible Help group

Documentation, AI Assistant, API Reference, and About are now inside a
collapsible sidebar group. State is persisted in `localStorage`. The group
is expanded by default.

### Grouped sidebar navigation

The flat link list is organized into six collapsible groups:

| Group | Contents |
|-------|----------|
| Fleet | Devices · CMDB · Agent Containers · Proxmox LXC · Virtualization · Network |
| Monitoring | Targets · Device Metrics · Listening Ports · Custom Scripts |
| Security | TLS/DNS Expiry · ACME Certificates · Patches · CVEs · Drift · Audit |
| Planning | Schedule · Calendar · Tasks · Maintenance · History |
| Admin | Settings · Users · API Keys · Library · Scripts · IaC Generator · Server Status |
| Help | Documentation · AI Assistant · API Reference · About |

Standalone top-level items: **Home** and **Links**.

Admin collapses by default; all other groups expand by default. State is
persisted per-group in `localStorage`.

## Bug Fixes

- **CSP `style-src-attr` violation** — inline `style="width:100%"` on the
  sidebar documentation table replaced with CSS class `isl-770`.
- **Modal delete buttons invisible** — `element.style.display = ''` was
  silently losing to a `d-none` class; fixed by explicitly setting `'block'`.

## Security

- Strict CSP (`default-src 'self'`, no `unsafe-inline`/`unsafe-eval`)
  maintained across all pages.
- Service-worker cache name bumped to `remotepower-shell-v3.1.0`.
