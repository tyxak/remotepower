# Links

**Fleet → Links** is a shared bookmark dashboard — a card grid, grouped by
category, of the web UIs and references your team actually uses (router
admin pages, NAS UIs, wikis, vendor portals). Every card opens in a new tab.

- **Internal links** (LAN-only, behind VPN, …) are amber-bordered;
  **external** links are plain — so you can tell at a glance whether a link
  will work from outside the network.
- Cards carry a name, URL, category and optional note; anyone with the
  links permission can add/edit.
- Operator-authored URLs are scheme-checked (http/https only) before they
  are rendered as clickable.
